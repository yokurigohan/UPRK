1.  Создаем базу данных в MySQL. 

    Запускаем базу данных с помощью команд: "create database Project_Data_Base;" и "use Project_Data_Base;". 

2.  
    В файлах API.py, bd.py, config.ini и main.py вставляем данные о своей базе данных в host, user, password и database.

    Пример: 
    host="127.0.0.1", 
    user="root", 
    password="Pa$$w0rd", 
    database="BNR" 
 
3.  Вставляем путь до файла access в файлах main.py и config.ini в "log_file" и путь до папки ProjectWork в файле config.ini в "log_path"
    Пример:
    log_file = X:/PMC/access.log
    log_path = X:/PMC
 
4.  Запускаем файл main.py 

5.  После того как он прогрузится полностью, переходим в MySQL и обновляем нашу базу данных, после чего вводим команду "select * from access_logs" чтобы просмотреть таблицу. 

6.  Теперь запускаем файл API.py

7.  У нас должна появиться ссылка: http://127.0.0.1:5000/ 
 
8.  Копируем её и вставляем в браузер (должна вывестись ошибка "not found"). 

9.  Добавляем к этой ссылке "/logs": http://127.0.0.1:5000/logs 
 
10. После этого у нас должны появиться все логи из файла "access.log". 

11. Копируем ссылку из пункта 10 и добавляем в неё "?start_date=YYYY-MM-DD&end_date=YYYY-MM-DD":
http://localhost:5000/logs?start_date=2010-01-01&end_date=2023-12-31 
 
12. После этого у нас должны появиться логи, по отсортированные по указанным датам. 

13. Копируем ссылку из пункта 10 и добавляем в неё "?ip=000.000.000.000": http://localhost:5000/logs?ip=146.1.1.2

14. После этого у нас должны появиться только те логи, у которых есть данный ip
